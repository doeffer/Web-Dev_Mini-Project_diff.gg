## APIs

- **Riot Games API** — Player data, match history, ranked stats
  https://developer.riotgames.com/docs/lol
  Endpoints used: Account-V1, Summoner-V4, League-V4, Match-V5

- **Data Dragon (Riot Games)** — Static game assets (champion images, profile icons, rank emblems)
  https://developer.riotgames.com/docs/lol#data-dragon
